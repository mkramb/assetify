Assets manager
========

**Installing**

    npm init

**Running tests**

    npm test

**Example usage**

    bin/cmd.js --baseDir=./test --entryPoint=./a.js
    bin/cmd.js --baseDir=./test --entryPoint=./a.js --outputTree
    bin/cmd.js --baseDir=./test --entryPoint=./a.js --outputJS=./bundle.js --outputCSS=./bundle.css
